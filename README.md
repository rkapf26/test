# Simple Web Project

This is a simple and basic web project designed to demonstrate a minimal HTML structure as a test for GitHub Pages.

## Demo

You can visit the deployed page here:
[https://your-username.github.io/simple-web-project/](https://your-username.github.io/simple-web-project/)
*(Replace `your-username` with your actual GitHub username)*

## Features

- Displays **Hello, GitHub!** in the center of the page.
- Minimal and lightweight HTML/CSS code.
- Designed to work with GitHub Pages using HTTPS.
